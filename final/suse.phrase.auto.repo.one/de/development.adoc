== Voraussetzungen:

* https://kind.sigs.k8s.io/[freundlich]
* https://helm.sh/[helm]
* https://tilt.dev/[Neigung]

== Erstellen einer lokalen Entwicklungsumgebung

. Klonen Sie das https://github.com/rancher/turtles[{product_name}] Repository lokal
. Erstellen Sie *tilt-settings.yaml*:
+
[source,yaml]
----
{
    "k8s_context": "k3d-rancher-test",
    "default_registry": "ghcr.io/turtles-dev",
    "debug": {
        "turtles": {
            "continue": true,
            "port": 40000,
            "insecure_skip_verify": "true"
        }
    }
}
----
+
. Öffnen Sie ein Terminal im Stammverzeichnis des Repositorys {product_name} 
. Führen Sie das Folgende aus:
+
[source,bash]
----
make dev-env

# Or if you want to use a custom hostname for Rancher and your ngrok credentials
NGROK_API_KEY=`key` NGROK_AUTHTOKEN=`token` RANCHER_HOSTNAME=my.customhost.dev make dev-env
----
+
. Wenn Sie keine ngrok-Zugangsdaten haben und sie nicht in einem früheren Schritt angegeben haben, können Sie auch ein neues Terminal öffnen und ngrok oder inlets starten, nachdem tilt gestartet wurde:
+
[source,bash]
----
kubectl port-forward --namespace cattle-system svc/rancher 10000:443
ngrok http https://localhost:10000
----

== Was passiert, wenn Sie `make dev-env` aufrufen?

. Ein Cluster https://kind.sigs.k8s.io/[der Art] wird unter Verwendung der folgenden https://github.com/rancher/turtles/blob/main/scripts/kind-cluster-with-extramounts.yaml[] erstellt.
. Die folgenden vorausgesetzten Komponenten werden mit Helm installiert:
 ** Rancher Manager.
 ** Zertifizierungsmanager.
 ** ngrok Kubernetes Operator.
. Docker-Images werden lokal erstellt, und Helm setzt die folgenden Controller im Cluster ein:
 ** Core Cluster API Controller.
 ** RKE2 Bootstrap/Control Plane Provider Steuerungen.
 ** Docker Infrastructure Provider Controller.
 ** CAPI-Add-on-Provider für Flottencontroller.
 ** {product_name} Controller.
 ** {product_name} Cluster-API-Operator-Controller.
 ** {product_name} ClusterClass Controller.
 ** {product_name} Tag 2 Betriebskontrolleur.
. Die Entwicklungsumgebung wird mit `tilt up` gestartet.
